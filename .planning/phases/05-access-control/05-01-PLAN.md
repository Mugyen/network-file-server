  <action>
Create `server/app/services/auth_service.py`:

1. `hash_password(plain_password: str) -> bytes`: Validate non-empty and <= 72 bytes, then `bcrypt.hashpw(plain_password.encode("utf-8"), bcrypt.gensalt())`. Raise ValueError for empty or >72 bytes.

2. `verify_password(plain_password: str, hashed_password: bytes) -> bool`: Validate non-empty plain_password (raise ValueError). Return `bcrypt.checkpw(plain_password.encode("utf-8"), hashed_password)`. bcrypt.checkpw is constant-time.

3. `class AuthTokenService`: Constructor takes `secret_key: str` (no default). Stores `self._serializer = URLSafeTimedSerializer(secret_key)`. Method `create_token() -> str`: returns `self._serializer.dumps({"authenticated": True})`. Method `validate_token(token: str) -> bool`: try `self._serializer.loads(token)` and return True, except `(BadSignature, SignatureExpired)` return False.

4. Module-level `_token_service: AuthTokenService | None = None` with `get_token_service() -> AuthTokenService` (raises RuntimeError if not set) and `set_token_service(service: AuthTokenService) -> None`. This follows the same pattern as config.py's get/set_server_config.

Write comprehensive tests in `server/tests/test_auth_service.py`.
