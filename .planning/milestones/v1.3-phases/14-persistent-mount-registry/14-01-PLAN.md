    3. Create `relay/app/services/sqlite_registry.py` with:
       - `ReclaimResult` frozen dataclass with `remaining_ttl: int`
       - `SqliteMountRegistry` class:
         - `__init__(self, db: aiosqlite.Connection)` storing `self._db` and `self._connections: dict[str, TunnelConnection] = {}`
         - `@classmethod async def create(cls, db_path: str) -> SqliteMountRegistry` — opens aiosqlite connection, sets `PRAGMA journal_mode=DELETE`, creates `mounts` table (code TEXT PRIMARY KEY, status TEXT NOT NULL, agent_ip TEXT NOT NULL, created_at REAL NOT NULL, expires_at REAL, ttl_warned INTEGER NOT NULL DEFAULT 0), creates indexes on agent_ip and status, runs `_startup_cleanup()`, logs INFO if DB file did not exist before connect ("No existing mount database -- starting fresh")
         - All public methods are `async`. Every method matches the MountRegistry API but is async and backed by SQLite for metadata, in-memory dict for connections.
         - `register()`: INSERT OR REPLACE into SQLite, store connection in `_connections`, commit. Validates code is non-empty (raise ValueError).
         - `deregister()`: DELETE from SQLite where code=?, remove from `_connections`. Raise MountNotFoundError if DELETE affects 0 rows. Commit.
         - `get_connection()`: SELECT status FROM mounts WHERE code=?. If no row, raise MountNotFoundError. If OFFLINE, raise MountOfflineError. If EXPIRED, raise MountExpiredError. Return `self._connections[code]`.
         - `mark_offline()`: SELECT status FROM mounts WHERE code=?. If no row, raise MountNotFoundError. If status is ONLINE: UPDATE status to OFFLINE, commit, and `self._connections.pop(code, None)`. If status is not ONLINE (already OFFLINE or EXPIRED): no-op (race guard — prevents late disconnect handler from undoing a reclaim).
         - `has_mount()`: SELECT 1 FROM mounts WHERE code=?, return bool.
         - `count_mounts_by_ip()`: SELECT COUNT(*) FROM mounts WHERE agent_ip=? AND status != 'expired'.
         - `active_mounts()`: SELECT all columns FROM mounts WHERE status != 'expired'. Return list of MountRecord, with `connection` field from `self._connections.get(code)` (None for OFFLINE mounts).
         - `try_reclaim()`: SELECT status, agent_ip, expires_at FROM mounts WHERE code=?. Return None if: no row, status != OFFLINE, IP mismatch, or expires_at <= now. On success: UPDATE status to ONLINE, commit, store connection in `_connections`, return ReclaimResult(remaining_ttl=int(expires_at - now)).
         - `delete_expired_before(cutoff: float)`: DELETE FROM mounts WHERE status='expired' AND expires_at IS NOT NULL AND expires_at < ?. Commit.
         - `_startup_cleanup()`: (a) DELETE expired records past 6h retention, (b) UPDATE newly-expired records to EXPIRED status, (c) UPDATE all ONLINE to OFFLINE. Commit. Log count of cleaned/marked records at INFO level.
         - `close()`: await self._db.close().
         - Use `time.time()` for all wall-clock comparisons (never time.monotonic).
