  <behavior>
    - Agent WebSocket accepts optional `ttl` query parameter (integer seconds)
    - If ttl exceeds config.max_ttl_seconds, relay caps it to the maximum (does not reject)
    - If ttl is None/omitted, relay uses config.max_ttl_seconds as the TTL
    - MountRecord.expires_at is set to created_at + capped_ttl
    - TTL sweep identifies mounts where time.monotonic() >= expires_at and status is ONLINE
    - TTL sweep sends ttl_warning message {"type": "ttl_warning", "expires_in": N} when mount is within warning_before_seconds of expiry (but only once -- track warned state)
    - TTL sweep marks expired mounts as EXPIRED and closes their WebSocket connection
    - TTL sweep runs on configurable interval (config.ttl_sweep_interval_seconds)
    - TTL sweep is started as asyncio background task in app lifespan, cancelled on shutdown
    - Sweep handles double-deregister race gracefully (deregister is already caught by MountNotFoundError in agent_ws finally block)
  </behavior>
  <action>
    1. Create `relay/app/services/ttl_sweep.py`:
       - `async def run_ttl_sweep(registry: MountRegistry, config: RelayConfig) -> None`:
         - Infinite loop: `await asyncio.sleep(config.ttl_sweep_interval_seconds)`.
         - Call `registry.active_mounts()` to get a snapshot of mounts.
         - For each mount with expires_at not None and status ONLINE:
           - Calculate remaining = mount.expires_at - time.monotonic().
           - If remaining <= 0: log at INFO "TTL expired: code={code}", set mount.status = MountStatus.EXPIRED, close the connection with `await mount.connection.close()`.
           - Elif remaining <= config.warning_before_seconds and mount has not been warned: send ttl_warning control message `{"type": "ttl_warning", "expires_in": int(remaining)}` via `await mount.connection.send_control(...)`. Track warned state by adding a `ttl_warned: bool` field to MountRecord (default False).
         - Wrap per-mount processing in try/except to prevent one bad mount from killing the sweep. Log exceptions at ERROR level.

    2. Add `ttl_warned: bool` field to MountRecord in mount_registry.py (default-like behavior: set to False in register()).

    3. Update `relay/app/main.py` create_relay_app():
       - Add FastAPI lifespan using `@asynccontextmanager`:
         ```python
         @asynccontextmanager
         async def lifespan(app: FastAPI):
             sweep_task = asyncio.create_task(
                 run_ttl_sweep(get_registry(), get_config())
             )
             yield
             sweep_task.cancel()
             try:
                 await sweep_task
             except asyncio.CancelledError:
                 pass
         ```
       - Pass `lifespan=lifespan` to `FastAPI(...)` constructor.
       - Import run_ttl_sweep from relay.app.services.ttl_sweep, get_config from relay.app.config, get_registry from mount_registry.

    4. Update `relay/app/routers/agent_ws.py`:
       - Add `ttl: int | None = Query(None)` parameter to agent_websocket (per user decision: "Agent specifies TTL as query parameter on WebSocket URL: /agent/ws?code=abc&ttl=3600").
       - Import get_config, time.
       - After WebSocket accept and code assignment:
         - `config = get_config()`
         - Cap TTL: `effective_ttl = min(ttl, config.max_ttl_seconds) if ttl is not None else config.max_ttl_seconds`
         - `now = time.monotonic()`
         - `expires_at = now + effective_ttl`
       - Extract agent IP from WebSocket headers: `agent_ip = websocket.headers.get("x-forwarded-for", "").split(",")[0].strip() or (websocket.client.host if websocket.client else "unknown")`
       - Pass agent_ip, created_at=now, expires_at=expires_at to registry.register().

       - Test sweep warning: create a mount expiring within warning_before_seconds, run sweep, verify ttl_warning control message sent and ttl_warned set to True.
       - Test sweep skips already-warned mounts: set ttl_warned=True, run sweep, verify no duplicate warning sent.

       - For unit tests of the sweep, call run_ttl_sweep indirectly by extracting the per-mount logic into a testable function, or mock asyncio.sleep to run a single iteration.
