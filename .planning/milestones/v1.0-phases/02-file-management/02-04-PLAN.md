       - Export function useFileSelection(files: FileEntry[]) returning { selectedNames: Set<string>, isAllSelected: boolean, isIndeterminate: boolean, toggleSelect: (name: string) => void, toggleSelectAll: () => void, clearSelection: () => void, selectedCount: number }
       - selectedNames stored in useState as Set<string>
       - toggleSelect: add or remove name from set
       - toggleSelectAll: if all selected, clear; otherwise select all
       - isAllSelected: selectedNames.size === files.length && files.length > 0

       - clearSelection: empty the set
       - Reset selection when files array changes (new directory)

       - downloadFile(path: string): void -- create a temporary <a> element with href="/api/files/download?path={encodeURIComponent(path)}" and download attribute, click it programmatically, remove it. Direct browser download, no confirmation per user discretion.
       - downloadAsZip(paths: string[]): Promise<void> -- POST /api/files/download-zip with body { paths }. Receive as blob, create object URL, trigger download via <a> element.
       - deleteFiles(paths: string[]): Promise<{ deleted: string[] }> -- DELETE /api/files with body { paths }. Use a new apiDelete<T> function added to client.ts.
       - renameFile(path: string, newName: string): Promise<{ path: string }> -- PATCH /api/files/rename with body { path, new_name: newName }. Use a new apiPatch<T> function added to client.ts.
       - createFolder(parentPath: string, name: string): Promise<{ path: string }> -- POST /api/folders with body { parent_path: parentPath, name }. Use a new apiPost<T> function added to client.ts.

    3. Extend client/src/api/client.ts:
